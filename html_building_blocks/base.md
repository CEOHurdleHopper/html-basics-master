# The base or <"base"> element 🏠

## What is it? 🤔
- The base element is the creator specified origin URL.
- This element is used to organize all other connected URLs to the origin and set the structure and pathways of the webpage.
- This allows a webpage to effectively use relative URLs or links that do not need absolute locational understanding but instead use a kind of shortened location that relys on the URLs position/connection to the base URL for pathway understanding.
- There can only ever be one base element in a webpage all others after the first are ignored.
- The base element requires either a link or a href element to function properly.
