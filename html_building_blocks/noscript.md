# The noscript or <"noscript"> element ❌📃

## What is it? 🤔
- This element is dependent on weather or not an individual user enables scripting using their browser.
- Todays browsers allow for users to disable scripting and leaves these choices to their needs.
- If scripting is enabled noscript element items will <ins>NOT</ins> appear, and if diabled they will appear.
